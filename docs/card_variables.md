# Frontend Component Card Variables

This document lists the variable names used for major card components in the application's pages. These variables represent the distinct sections of content within each page.

## Bible Page (`Bible.tsx`)
- **Passages List**: `B_Card_01`
- **Passage Details**: `B_Card_02`

## Quotes Page (`Quotes.tsx`)
- **Quotes List**: `Q_Card_01`
- **Quote Details**: `Q_Card_02`

## Start Here Page (`StartHere.tsx`)
- **Welcome & Intro**: `SH_Card_01`
- **Application Features**: `SH_Card_02`

## Thoughts Page (`Thoughts.tsx`)
- **Thoughts List**: `TC_Card_01`
- **Thought Details**: `TC_Card_02`

## Topics Page (`Topics.tsx`)
- **Topics List**: `T_Card_01`
- **Topic Details**: `T_Card_02`

## Tags Page (`Tags.tsx`)
- **Tags List**: `TG_Card_01`
- **Tag Items List**: `TG_Card_02`
- **Content / Description**: `TG_Card_03`
